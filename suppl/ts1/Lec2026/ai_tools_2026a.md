### AI tools
Recommended AI tools for the course:
* Web-based AI assistants:
  - Claude, Gemini, ChatGPT, Grok
* Agentic CLI tools: 
    - Claude CLI (with), Gemini CLI, Codex CLI (by ChatGPT), Grok CLI
- Agentic AI with Hebrew support: 
    - Claude Code
    - Codex App for Windows (released 04-03-26)
* Tools with free(?) access for students:
  * Web-based: [Perplexity](https://www.perplexity.ai/)
  * Code-oriented: [GitHub Copilot](https://copilot.github.com/) and GitHub Copilot CLI (reduced functionality for students starting, policy updated 12-03-26)
